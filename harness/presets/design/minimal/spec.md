# minimal 디자인 프리셋

> **영역**: Design
> **버전**: 0.1.0
> **기준일**: 2026-04-21

---

## 1. 개요

뉴트럴 팔레트·넓은 여백·플랫한 시각을 지향하는 디자인 프리셋. client 디자인 요구가 부족하거나 "깔끔하고 절제된" 톤이 필요할 때 designer 에이전트가 제안.

## 2. 제공하는 것 (Included)

- **컬러**: 그레이스케일 중심 + 단일 accent (딥 슬레이트)
- **타이포**: Inter (sans-serif), 제목/본문 폰트 동일
- **레이아웃**: 12-컬럼 그리드, 풍부한 여백(예: section 간 96px)
- **시각 스타일**: 플랫, 그림자 거의 없음, 얇은 1px 보더
- **라운드**: 2~4px (단단한 느낌)
- **모션**: 절제된 fade/slide, 150~200ms

## 3. 적합한 경우

- B2B SaaS 관리자 페이지
- 콘텐츠 중심 웹 (블로그, 문서, 포트폴리오)
- 브랜드 색이 약하거나 중립 이미지가 중요한 경우

## 4. 부적합한 경우

- 소비자 대상 엔터테인먼트·게이밍 → `playful` 권장
- 전통·신뢰 강조 기업 브랜드 → `corporate` 권장
