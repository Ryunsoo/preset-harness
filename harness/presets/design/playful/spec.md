# playful 디자인 프리셋

> **영역**: Design
> **버전**: 0.1.0
> **기준일**: 2026-04-21

---

## 1. 개요

비비드 팔레트·큰 라운드·부드러운 그림자·활발한 모션을 지향하는 프리셋. 소비자 대상 서비스·젊은 브랜드·엔터테인먼트에 적합.

## 2. 제공하는 것 (Included)

- **컬러**: 비비드 핑크·바이올렛·선옐로우 기반 + 소프트 뉴트럴
- **타이포**: Nunito (rounded sans-serif)
- **레이아웃**: 12-컬럼 그리드, 넉넉한 여백
- **시각 스타일**: 부드러운 그림자, 미세 그라디언트 허용
- **라운드**: 12~24px (활기찬 느낌)
- **모션**: spring 기반, 250~350ms, 장식적 마이크로 애니메이션 허용

## 3. 적합한 경우

- 소비자 대상 모바일 앱·서비스
- 커뮤니티·소셜·크리에이터 플랫폼
- 교육·아동 서비스, 이벤트·캠페인 페이지

## 4. 부적합한 경우

- 금융·공공·엔터프라이즈 톤 → `corporate` 권장
- 절제된 콘텐츠 중심 → `minimal` 권장
