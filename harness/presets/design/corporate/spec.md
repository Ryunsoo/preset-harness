# corporate 디자인 프리셋

> **영역**: Design
> **버전**: 0.1.0
> **기준일**: 2026-04-21

---

## 1. 개요

정형 그리드·신뢰감·전문성을 지향하는 기업형 디자인 프리셋. 금융·B2B 엔터프라이즈·공공 분야의 client 요구에 대응하기 위한 기본 제안안.

## 2. 제공하는 것 (Included)

- **컬러**: 네이비·스틸 블루 주도 + 중립 그레이 기반
- **타이포**: IBM Plex Sans (본문) / IBM Plex Serif (제목 강조 옵션)
- **레이아웃**: 12-컬럼 그리드, 일관된 section padding
- **시각 스타일**: 중간 톤 그림자, 명확한 카드 경계
- **라운드**: 4~8px (정돈된 느낌)
- **모션**: 차분한 fade/slide, 200~250ms

## 3. 적합한 경우

- 금융·보험·법무 client
- B2B 엔터프라이즈 제품, 내부 인사관리·ERP
- 공공기관·기업 포털

## 4. 부적합한 경우

- 소비자 엔터테인먼트·게이밍 → `playful` 권장
- 극도로 절제된 미니멀 요구 → `minimal` 권장
